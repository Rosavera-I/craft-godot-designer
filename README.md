# craft-godot-designer

CRAFT harness cartridge for godot-designer expertise.

## Usage

Install via CRAFT CLI:
```bash
craft harness install github:JMoak/craft-godot-designer
```

## Files

- `craft.toml` — Manifest and metadata
- `prompts/system.md` — System prompt
- `memory/schema.toml` — Memory schema
- `tools/mcp.toml` — MCP tool definitions
- `validators/checks.tdd` — TDD validation checks
